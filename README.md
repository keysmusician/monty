## About
The Monty Btyecode Interpreter is a **command interpreter** for **Monty bytecode.** **Monty 0.98** is the scripting language that is compiled into Monty bytecodes.

<!-- ## Usage -->

## Opcodes
* `add`
* `nop`
* `pall`
* `pint`
* `pop`
* `push`
* `swap`
